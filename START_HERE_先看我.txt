这是 ORA 外送周报“云端服务器版”项目，不是直接双击 index.html 使用的静态网页。

如果只是你自己电脑测试：
1. 双击 本地启动_windows.bat
2. 等命令窗口显示 http://127.0.0.1:8000
3. 浏览器会自动打开看板
4. 上传 15 个表格，点击“生成周报”

如果要发给别人一个链接使用：
1. 把整个 ora-report-cloud 文件夹上传到 GitHub 仓库
2. 在 Render / Railway / 公司服务器 上部署这个项目
3. 部署成功后会得到一个公网网址，例如 https://xxx.onrender.com
4. 把这个公网网址发给别人

注意：
- file:// 本地文件地址只能你自己电脑打开，别人不能用。
- 云端部署后，别人的文件会上传到服务器处理；如果数据敏感，建议部署在公司账号或公司内网服务器。
- 最新评分表不需要上传。
